# AGENTS

- 每次开始任务先读根 `README.md`。
- 这是公开发布仓库，只维护 `README.md`、`index.html`、`release-notes/`、`update/`、截图和品牌素材等公开资产。
- 不在这里引入私有源码、部署脚本、PM2、Prisma、数据库迁移或远端 hook 约定。
- 默认工作流是：本地修改、自检、提交、推送；不要写“自动部署”说明。
- 改双语文案时中英文必须同步；改版本说明或更新清单时，`README.md` 链接与公开产物口径必须一致。
- 内容尽量精简，优先复用已有公开素材，保持高内聚、低耦合。
- 功能或文档变更时，同步更新对应说明文件。
